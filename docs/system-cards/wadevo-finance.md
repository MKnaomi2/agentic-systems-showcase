# Wadevo Finance system card

- Status: live invite-only alpha with a public landing page
- Period: January through July 2026
- Ownership: end-to-end product design and implementation
- Public surface: https://app.wadevo.com
- Source: private; walkthrough available to hiring teams

## Core decision

A deterministic attention engine owns alerts and projections. Bounded AI
interprets natural-language questions over scoped data rather than becoming the
source of financial truth.

## Publicly inspectable

The public landing page, TLS boundary, authentication boundary, public
architecture, and the sanitized reference patterns in this repository. The
current alpha uses manual entry and statement upload; bank sync is not enabled.

Implementation details, tests, repository history, and a source walkthrough.
