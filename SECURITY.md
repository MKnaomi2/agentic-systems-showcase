# Security policy

This repository is intentionally dependency-free at runtime and contains only
public reference code and a sanitized portfolio snapshot.

Report a suspected vulnerability privately to
[alvento.lisp@proton.me](mailto:alvento.lisp@proton.me). Do not include live
credentials, private financial data, or health data in a report.

The reference authority engine demonstrates policy patterns. It is not a
drop-in authentication or authorization product and has not been independently
audited.
